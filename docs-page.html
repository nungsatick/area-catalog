เทคนิคการขึ้นชุดข้อมูล

เป็นส่วนสนับสนุนให้หน่วยงานสามารถนำขึ้นชุดข้อมูลเข้าสู่บัญชีข้อมูลจังหวัด ให้เป็นไปตามแนวทางและมาตรฐานเดียวกัน

การจัดทำบัญชีข้อมูล

เนื้อหาสนับสนุนการจัดทำชุดข้อมูลจังหวัด และการนำขึ้นระบบบัญชีข้อมูลบนแพลตฟอร์ม CKAN Open-D

# หัวข้อ
1การเลือกภารกิจและกระบวนงาน
2รายละเอียดคำอธิบายชุดข้อมูล
3การกำหนดชุดข้อมูล
4การจัดการแท็กเพื่อการค้นหา
5การกำหนดผู้เป็นเจ้าของข้อมูล
6การกำหนดองค์กร
7การนำองค์กรขี้นระบบ
8การกำหนดหมวดหมู่ของชุดข้อมูล
9การนำหมวดหมู่ขึ้นระบบ
10การนำชุดข้อมูลพร้อมคำอธิบายชุดข้อมูล (Metadata) ขึ้นระบบ
11การระบุลิงก์ (Link) แหล่งข้อมูลต้นทาง
12การค้นหาชุดข้อมูล
13เอกสารและวิดีโอประกอบเพิ่มเติม
14สื่อผสมการจัดทำระบบบัญชีข้อมูล
15บทความสั้น สนับสนุนการจัดทำระบบบัญชีข้อมูล

การลงทะเบียนบัญชีข้อมูล

การลงทะเบียนบัญชีข้อมูลเข้าสู่ระบบบัญชีข้อมูลภาครัฐ (GD Calatalog) เพื่อนำไปสู่การเปิดเผยข้อมูลภาครัฐและใช้ประโยชน์ข้อมูลร่วมกัน

# หัวข้อ
1ผู้ที่เกี่ยวข้องในการลงทะเบียนบัญชีข้อมูล
2ขั้นตอนการลงทะเบียนบัญชีข้อมูล
3การสร้างบัญชีผู้ใช้บนระบบ Open ID
4แนวทางการลงทะเบียนบัญชีข้อมูลภาครัฐ
5คู่มือการใช้งานสำหรับผู้จัดการระบบบัญชีข้อมูลหน่วยงาน
6คู่มือการใช้งานเจ้าหน้าที่ลงทะเบียนชุดข้อมูล
7คำแนะนำการตรวจสอบความถูกต้องก่อนการลงทะเบียนบัญชีข้อมูล
